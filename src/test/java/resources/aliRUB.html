      Как загрузить плату PI3 LTS?


      (1) загрузка от EMMC:

      У нас есть предустановленное изображение android на emmc. Вы можете зарядить блок питания без sd-карты непосредственно для загрузки платы.


      (2) Загрузка с sd-карты:

      Подготовьте новую 8 Гб выше, sd-карту sandisk класса 10.

      Затем загрузите изображение linux с веб-сайта orange pi, сначала сжечь изображение linux на sd-карту.

      Затем вставьте sd-карту, сгоревшую с изображением на доску.

      Наконец, зарядка питания и другие компьютерные аксессуары.
